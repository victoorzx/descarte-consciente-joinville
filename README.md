# Sistema de Apoio ao Descarte Consciente - Joinville

## Sobre o Projeto
Este projeto foi desenvolvido para a disciplina de Fábrica de Software (Projeto de Nivelamento). O objetivo é fornecer uma solução digital voltada ao cidadão de Joinville para organizar informações de descarte de resíduos, facilitando a consulta de ecopontos, solicitação de coleta de grande porte e registro de denúncias de descarte irregular.

## Tecnologias Utilizadas
* **Front-end:** HTML5, CSS3, Bootstrap 5
* **Lógica e Persistência:** JavaScript Vanilla utilizando a API nativa `LocalStorage` para simular um banco de dados NoSQL no lado do cliente.
* **Prototipação:** Figma
* **Gestão de Tarefas:** Trello

## Como Executar
O sistema foi desenvolvido seguindo a arquitetura Single Page Application (SPA) estática. Para executá-lo, não é necessário rodar um servidor local complexo.
1. Clone este repositório.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` diretamente em qualquer navegador moderno (Chrome, Edge, Firefox).
